Create a dynamic e-commerce product catalogue for a web application where users can browse, filter, and sort products seamlessly. The project emphasises key industry practices such as API integration, responsive design, and advanced UI/UX enhancements for a real-world scenario. Creating a responsive and visually appealing layout and products for a real-world application. It should have the following and any other necessary enhancement.

1. Product Card
Purpose: Showcase product image, name, price, rating, and CTA (e.g. “Add to Cart”)

Layout Logic:

Frame size: 300x400 (adjustable for responsiveness)

Auto Layout: Vertical

Elements:

Image (top, 60% height)

Title + Price (stacked)

Rating (stars or numeric)

Button (primary CTA)

Enhancements:

Hover variant with shadow and scale

“Sale” or “New” badge component

Wishlist icon (heart) in top-right corner

🧮 2. Filter Panel
Purpose: Let users refine product results by category, price, rating, etc.

Layout Logic:

Sidebar frame or modal

Auto Layout: Vertical

Elements:

Category checkboxes

Price range slider

Rating filter (stars or dropdown)

Apply/Reset buttons

Enhancements:

Collapsible sections

Sticky positioning for persistent access

Animated transitions for opening/closing

🔽 3. Sort Dropdown
Purpose: Sort products by relevance, price, rating, etc.

Layout Logic:

Frame: Dropdown with label + icon

Auto Layout: Vertical list of options

Variants: Default, Open, Selected

Enhancements:

Keyboard accessibility

Icons beside options (e.g. ↑↓ for price)

🧭 4. Header / Navbar
Purpose: Brand identity + global navigation

Layout Logic:

Frame: 1440x80 (desktop), responsive with Auto Layout

Elements:

Logo (left)

Nav links (center or right)

Cart icon, profile icon (right)

Enhancements:

Sticky on scroll

Mobile variant with hamburger menu

Search bar integration

📚 5. Sidebar
Purpose: Secondary navigation or filter access

Layout Logic:

Frame: 300px width

Auto Layout: Vertical

Elements:

Section headers

Nav links or filter toggles

Enhancements:

Light/dark mode variants

Expand/collapse animation

Icons for each section

🦶 6. Footer
Purpose: Site-wide links, contact info, legal

Layout Logic:

Frame: 1440x200 (desktop)

Auto Layout: Horizontal sections

Elements:

About, Help, Contact

Social icons

Newsletter signup

Enhancements:

Responsive stacking on mobile

Subtle background gradient or texture
